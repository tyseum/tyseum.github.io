걱정마세요! 입력한 정보와 인감 이미지는 저장되지 않아요. PDF 생성과 다운로드에만 잠시 사용됩니다.

Guide

제출 서류 안내

이 사이트에서 작성 지원 계약서류, 착수서류, 완료서류, 청구서류 중 반복 작성이 필요한 양식을 자동으로 채워 PDF로 저장할 수 있습니다.

1 계약서류

용역금액이 500만원 이상인 경우에는 나라장터를 통해 계약을 진행합니다.

이 사이트에서 작성
  • 공사(용역·물품) 표준계약서나라장터 계약 시 생략 가능
  • 입찰유의서, 일반·특수조건나라장터 계약 시 생략 가능
  • 계약보증증권 또는 계약보증금 지급각서1천만원 미만은 지급각서 대체 가능
  • 청렴계약이행서약서서식 제공
  • 수의각서서식 제공
  • 용역계약 이행 승낙사항서식 제공
  • 조세포탈서약서서식 제공
  • 안전보건관리준수서약서서식 제공
  • 수의계약 체결 제한 여부 확인서서식 제공
  • 견적서 및 견적내역서이 사이트에서 작성
별도 제출
  1. 사업자등록증 사본
  2. 자격 면허, 등록증 사본 (해당 시)
  3. 개인/법인 인감증명서 원본
  4. 법인등기부등본 원본 (법인의 경우)

* 이 페이지에서 작성한 계약서류는 하나의 PDF로 저장한 뒤 나라장터 계약사이트에 직접 업로드합니다.

2 착수 서류

착수계와 참여자 관련 서류를 함께 준비합니다.

이 사이트에서 작성
  • 착수계2부 준비
  • 현장대리인계현장대리인 지정 시
  • 참여자명단현장대리인 외 참여자가 있을 시
  • 재직증명서현장대리인·참여자 필요 시
  • 예정공정표계약기간 기준 작성
  • 착수내역서계약금액 기준 작성
  • 보안각서참여자별 필요 시

3 완료 서류

용역 완료 후 제출하는 서류입니다.

이 사이트에서 작성
  • 완료계계약담당자 확인 날인 필요
  • 납품확인서과업지시서상 성과품 촬영사진 포함

4 청구 서류

대금 청구 시 필요한 서류입니다.

일부 작성
  • 청구서서식 제공
  • 하자보증증권 또는 하자보증금 지급각서해당 시
별도 제출
  1. 국세/지방세 납세 증명서
  2. 4대보험완납증명서
  3. 세금계산서 (hi@tyseum.or.kr 메일 발송)
  4. 통장사본
  5. 지역개발공채 (2,000만원 이상의 계약)

Main

계약 기본정보 입력

입력할 정보 선택 계약 기본정보를 먼저 입력하면 계약서류 PDF에 자동 반영됩니다.
업체정보

사업자등록증 기준으로 입력해주세요.

계약정보

계약명과 금액은 모든 서류에 반복 반영됩니다.

인감 이미지

인감은 선택사항입니다. 등록하면 PDF에 반영되고, 등록하지 않으면 출력 후 직접 날인하면 됩니다.

인감증명서상 인감
미등록, 사용인감계에서 직접 날인 가능
사용인감
미등록, 출력 후 직접 날인 필요

인감증명서상 인감과 실제 사용할 인감을 각각 등록할 수 있습니다. 사용인감은 각 서류 하단 날인 위치에 반영되고, 인감증명서상 인감은 사용인감계의 비교 칸에만 반영됩니다.

착수·완료·수행인력

현장대리인을 먼저 입력하고, 현장대리인 외 참여자가 있을 때만 참여자를 추가해주세요.

청구 정보

완료 후 청구서에 들어갈 계좌 정보를 입력해주세요.

예정공정표

과업내용별 수행 구간을 입력하면 예정공정표에 막대형 일정표로 반영됩니다.

공기(일)를 기준으로 적절한 기준과 칸 수를 추천합니다.

과업내용 시작 종료 보고/비고
견적서(시뮬레이션)

총사업비를 입력하면 공급가액과 부가세를 계산하고, 인건비·운영비 세부내역을 견적서 하단에 함께 출력합니다.

총사업비
공급가액0원
부가세0원
인건비
세부내역수량1수량2수량3단가
운영비
세부내역수량1수량2수량3단가
A. 소계(인건비+운영비)0원
일반관리비는 인건비와 운영비 합계(A)의 5% 이내로 입력할 수 있습니다.
기업이윤은 인건비·운영비·일반관리비 합계(A+B)의 10% 이내로 입력할 수 있습니다.
D. 최종 공급가액0원
최종 공급가액은 인건비, 운영비, 일반관리비, 기업이윤 합계(A+B+C)입니다.

일반관리비와 기업이윤을 입력하면 최종 차액이 계산됩니다.

최종 차액(E)이 0원이 되면 총사업비와 견적금액이 일치합니다.
